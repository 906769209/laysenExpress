# 前言


初学node，简单的用express做了一个自己的博客

博客地址：<a href="http://express.91laysen.cn/" target="_blank">


## 技术栈

node + express + mysql + swig


## 项目运行

#### 注意：，node 需要 6.0 以上版本 

```

npm install

node app.js

```
